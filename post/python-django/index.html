初始化

使用命令django-admin startproject HelloWorld创建一个名为Hello World的项目

使用模板和静态文件夹

在HelloWorld的根目录创建templates和statics文件夹

在settings.py里设置TEMPLATES目录'DIRS': [os.path.join(BASE_DIR,"templates")],

底部加入STATICFILES_DIRS=[os.path.join(BASE_DIR,"statics")]加入statics目录
