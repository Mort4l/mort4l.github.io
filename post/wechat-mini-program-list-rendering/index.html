小程序列表渲染

By Makki on May 27, 2021

小程序列表渲染(for循环)

code

<view wx:for="{{[1,2,3,4]}}">{{index}}哈哈哈{{item}}</view>

out

0哈哈哈1
1哈哈哈2
2哈哈哈3
3哈哈哈4

code

<view wx:for="{{[1,2,3,4]}}" wx:for-index="i" wx:for-item="content">{{i}}哈哈哈{{content}}</view>

out

0哈哈哈1
1哈哈哈2
2哈哈哈3
3哈哈哈4

code

<view wx:for="{{[{name:'dava',age:18},{name:'mike',age:29}]}}">name:{{item.name}}——age:{{item.age}}</view>

out

name:dava——age:18
name:mike——age:29
